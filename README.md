### 2) Learning Fourier Neural Operator (FNO) for PDEs  
**Scientific Machine Learning & Operator Learning**

ðŸ“Œ Repository:  
https://github.com/himanshuvnm/Learning_Fourier_Neural_Operator  

This project reproduces and analyzes Fourier Neural Operators (FNOs) for learning solution operators of partial differential equations. I study the learning dynamics and generalization behavior of FNOs on the 1D viscous Burgers' equation at different spatial resolutions.

**Key themes:**
- Operator learning for PDEs  
- Spectral neural networks  
- Scientific machine learning  
- Physics-informed modeling  
- Generalization across discretizations  

This work connects modern deep learning with classical numerical analysis and computational physics.
